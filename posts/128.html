MongoDB 官方图形化界面工具 MongoDB Compass 安装配置教程（Windows版）

本文最后更新于：2026年6月4日

MongoDB Compass 是 MongoDB 官方推出的免费、跨平台图形化管理工具，无需命令行即可可视化管理 MongoDB 数据。它支持连接本地/远程实例与 Atlas 云服务，提供数据浏览、文档编辑、可视化查询、索引管理、性能监控、数据导入导出等功能，还能分析数据模式、排查异常、优化查询，内置 MongoDB Shell 方便混合操作。

以下是 Windows 环境下安装 MongoDB Compass 安装配置教程：
